# Keputusan-Menggunakan-Metode-Fuzzy-Mandani-(The Bontot)

Fuzzy adalah sebuah logika matematika yang memberikan nilai dengan derajat kepercayaan. Dalam prosesnya Fuzzy Logic  akan mengolah data menjadi keputusan yang aturannya ditentukan oleh manusia sehingga terdapat nilai kemanusiaan. 

Program ini digunakan untuk memberikan skor ekologis terbaik berbasis fuzzy logic pada Porvins, yaitu dimana luas lahan Hutan yang ditutup lebih besar dari pada lahan Non Hutan. Dengan justifikasi, jika banyak lahan Hutan yang ditutup maka baik untuk alam, sedangkan jika lahan Non Hutan banyak yang ditutup maka akan ada kemungkinan kekurangan lahan untuk pemukiman, industri, dll.
